FoxBound Animal Detector

This application runs object detection algorithm on images to detect the presence
of animals in the images.
Please begin the application by clicking the start button below.

Start



Instructions

Step 1: Start

Start the application by clicking on the Start button in the top right corner.

Step 2: Upload

Upload images/zip of images along with email address and click submit.

Step 3: Wait

Wait for the model to process your images. You will receive notifications via email to track your progress.

Step 4: Downlaod

Once complete, you will receive another email notification with the download link which is valid for only 1 week.

Troubleshoot

If your download link doesn't work, check if a week has passed. If not, try again and if the problem persists, notify CDS.

If you entered your email incorrectly, submit the same batch for processing again with the correct email address.
